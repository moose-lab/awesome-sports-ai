# Code of Conduct

Awesome Sports Collection is intended to be a welcoming, practical, and professional community resource.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent while still giving clear feedback.
- Focus discussions on the quality, relevance, and usefulness of contributions.
- Give credit where appropriate.
- Respect different backgrounds, experience levels, sports communities, and technical viewpoints.

## Unacceptable Behavior

- Harassment, intimidation, or personal attacks.
- Discriminatory language or behavior.
- Sexualized language, imagery, or attention.
- Trolling, repeated disruption, or bad-faith arguments.
- Publishing private information without permission.
- Spam, self-promotion, or affiliate promotion that does not serve the list.

## Enforcement

Maintainers may remove comments, close issues, reject pull requests, or restrict participation when behavior harms the project or its contributors.

If you see behavior that violates this code, report it to the maintainers through GitHub issues or another maintainer-designated contact path. Reports will be reviewed with care and handled as privately as practical.

## Scope

This code applies to project spaces including issues, pull requests, discussions, and any other official project communication channels.
